== Description ==

<ul>
<li>Ever been tired of finding the best tags for your blogs?</li>
<li>Ever used autoblogger and felt unsatisfied with the internal tagging mechanism?</li>
<li>Ever had dumb editors that does not tag their posts or tag them poorly?</li>
</ul>

Then this is a plugin for you! Enter your Open Calais api key (it takes 1 min to get one) and you dont have to worry about tagging anymore. Used in combination with <a href="http://www.autowordpress.biz/">WP-Auto Trackback Sender</a> its a 'deadly' tool. You can't get more easy SEO than that!

Just install it and configure it through the plugin interface located in the Settings panel. As you publish or edit a post, the plugin gathers and sets the tags suitable for your post based, on your content.

== Installation ==

1. Upload the folder wp-auto-add-tags to the /wp-content/plugins/ directory
2. Activate the plugin WP-Auto Add Tags through the 'Plugins' menu in WordPress
3. Configure the plugin
4. Edit or write posts.
